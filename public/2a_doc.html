Purpose: The Brisbane Living Index is a page that represents the quality of life in a particular suburb or region by taking full advantage of the JavaScript-based map engine to retrieve and process datasets provided by the Brisbane City Council. Web application. Intuitive explanation.

Interactive method: This website is designed to be suitable for both responsively designed desktop and mobile devices. When a user opens a web page, there is a large card in the center of the page with a title and introduction that explains how to use the web page. Clicking or tapping any point in a Brisbane virtual location will bring up a small pop-up window showing the calculated index of the specified location and some brief descriptions. If you need more detailed information, click on the description and a small box will expand to show more detailed static information about the location.

Usage of datasets: The five indexes are calculated using records from the Brisbane City Plan data source and the crime rate of the department provided by the Queensland Police Service. The method used to calculate these indicators is based on several international standards such as the Noise Index, the Wildfire Index, and the Air Quality Index. Indexes for which no criteria are accepted are calculated based on the percentage of the national ranking.

Target Audience: Someone who wants to move somewhere in Brisbane. Living conditions are human primary needs, so anyone who wants to find good accommodation can benefit from this aspect. There are many attachments, so those who want to buy real estate will be most interested.
